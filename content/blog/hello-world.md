Welcome to my new blog. Posts here are just Markdown files in the repo —
no database, no CMS. To publish a new post, add a `.md` file to
`content/blog/`, commit, and push.

## Why Markdown?

- Simple: write in a text editor, no admin UI to maintain
- Free: no database or hosting cost beyond what Vercel already provides
- Versioned: every post's history lives in git
